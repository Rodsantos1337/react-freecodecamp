## React Fundamentals — Hands-On Coursework

This repository documents my journey through the [freeCodeCamp React course](https://www.youtube.com/watch?v=x4rFhThSX04\&t=377s), where I'm building a strong foundation in React by coding every project from scratch.

### What's Inside

A progressive collection of React projects and exercises covering:

- Core React: JSX, components, props, composition, declarative UI

- Data-Driven UIs: `.map()`, reusable components, props patterns, destructuring

- State Management: `useState`, complex state (arrays/objects), forms, conditional rendering, lifting state

- Side Effects: `useEffect`, data fetching, cleanup functions, refs

- Capstone Projects:

  - Tenzies — A dice game with state-driven UI, hold/roll logic, and accessibility improvements
  - Assembly: Endgame — A word-guessing game with keyboard input, game state management, and a11y-first design
